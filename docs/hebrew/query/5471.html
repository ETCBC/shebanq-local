Description

A query to locate texts with the words (in any order) "delivered from the hand," as part of a study of Exodus 2:15b-20. This query found textual links to Exodus 3, and also to Exodus 18, where Jethro uses the same phrase. It seems as if he never forgot his initial encounter with Moses! It also provides an important link with the Song of Moses in Deuteronomy 32.
