Description

Deze query zoekt naar zinnen in de Psalmen waar een wayyiqtol-vorm voorafgegaan wordt door een predicaat (in dit geval is het predicaat een phrase). Het voorafgaande predicaat is namelijk bepalend voor de betekenis van de wayyiqtol. De query zoekt op 'verse' en niet op 'sentence' om een zo groot mogelijk bereik te hebben. Soms staat het eerste predicaat namelijk wel in hetzelfde vers, maar in de voorafgaande 'sentence'. Deze query prentendeert niet de volledigheid van alle wayyiqtols die op een predicaat-phrase volgen te bevatten. Het eerste predicaat zou namelijk ook in één of twee voorafgaande zinnen kunnen staan, en zodoende buiten de query-resultaten vallen. Deze qeuery biedt wel voldoende resultaten om verder te analyseren.

Query instruction (mql)

SELECT ALL OBJECTS WHERE
[book book in (Psalmi)
  [verse
    [phrase FOCUS function = Pred OR function = PreO OR function = PreS]
    ..
    [word FOCUS lex = 'W']
    [word FOCUS vt = wayq]
 ]
]
