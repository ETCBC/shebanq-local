Description

Lam. 3:13 uses the expression "sons of his quiver (בְּנֵי, אַשְׁפָּתוֹ) which is an hapax legomenon. In such expression, חֵץ (arrow) could have been used instead of בֵּן (son). Therefore, this query searches for the occurrences of חֵץ (arrow) and אַשְׁפָּה (quiver) in a clause.
