Description

Deze query zoekt naar alle zinnen in de Psalmen waarin een wayyiqtol voorafgegaan wordt door een clause met een perfectum. Deze clause mag echter geen lijdend voorwerp of complement bevatten, zodat perfecti met intransitieve betekenis boven water komen.

Query instruction (mql)

SELECT ALL OBJECTS WHERE
[book book in (Psalmi)
  [verse
    [clause
      NOTEXIST [phrase function = Objc]
      ..
      NOTEXIST [phrase function = Cmpl]
      ..
      [phrase function = Pred OR function = PreO OR function = PreS
        [word FOCUS vt=perf]
      ]
      ..
      NOTEXIST [phrase function = Objc]
      ..
      NOTEXIST[phrase function = Cmpl]
    ]
    ..
    [word lex = 'W']
    [word FOCUS vt = wayq]
 ]
]
