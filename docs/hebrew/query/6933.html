Description

This query returns all results in which a donkey/ezelin speaks. First, the results in which the donkey/ezelin is the subject of speaking were searched. This yields two results (Numbers 22). Then a search was made for those times when speaking is associated with the donkey/ezelin. This yields 3 (false) results (1 Samuel 10:14 and 2 Kings 4:22 and 24).

Query instruction (mql)

select all objects where
// First look for those times when the donkey/ezelin is the subject of speech and the other person is the object.
[clause
  [UnorderedGroup
    [phrase function=Subj            //Subject must be the donkey.
      [word FOCUS lex = '>TWN/']
    ]
    [phrase function=Pred            //word function must be predicate and must be יֹּאמֶר֩or דבר. In addition, it must be a wayyiqtol.
      [word focus (lex = '>MR['
                and vt = wayq
                  )
               or (lex = ' DBR/'
                and vt = wayq
                  )
      ]
    ]
  ]
]

or
// From here also search on the verb speak, without specified subject.
// Note that the section below gives approximate results.  
[clause
  // Search on a phrase in which the donkey/ezelin is subject.
  [UnorderedGroup
    [phrase function = Objc
      [word focus lex = '>TWN/']
    ]
  ]
]
// The phrases should be immediately followed by the verb 'Speaking', which should be a wayyiqtol.
[clause typ = Way0
  // Search on a phrase without a complement.
  [phrase function = Pred
    [word focus (lex = '>MR['
              and vt = wayq
                )
             or (lex = 'DBR/'
              and vt = wayq
                )
    ]
  ]
]

or
// From here also search on the verb speak, without specified subject. In different order from above.
// Note: the section below gives approximate results.  
[clause typ = Way0
  // First search on a phrase without a complement.
  [phrase function = Pred
    [word focus (lex = '>MR['
              and vt = wayq
                )
             or (lex = 'DBR/'
              and vt = wayq
                )
    ]
  ]
]
[clause
  // Search on a phrase in which the donkey/ezelin is subject.
  [UnorderedGroup
    [phrase function = Objc
      [word focus lex = '>TWN/']
    ]
  ]
]
