Description

This query selects all occurrences of the lexemes "קֹדֶשׁ" (CQD/) and "מִקְדָּשׁ" (MCQD/) functioning as grammatical objects in Biblical Hebrew. It explores how holiness—both as abstract sanctity (qodesh) and as sacred space (miqdash)—is syntactically governed. The goal is to identify:

Contexts in which holiness is acted upon or affected

Whether divine, human, or cultic agents typically act on holiness

This analysis contributes to understanding the theological role of holiness in texts like Numbers 17, where priestly legitimacy is marked by divine holiness.
