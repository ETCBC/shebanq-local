Description

The objective is to find a clause with a phrase which functions as a subject combined with a passive participle as predicative complement, parallel to the phrase found in Nehemia 1:3 וְחוֹמַ֤ת יְרוּשָׁלִַ֨ם֙ מְפֹרָ֔צֶת, and Nehemia 1:6 וְֽעֵינֶ֪יךָ פְתֻוּח֟וֹת.

This in order to assist with the translation of this clause in that verse.
