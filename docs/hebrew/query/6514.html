Description

Do the nations elsewhere receive inheritance (“נַחֲלָה”) by the Lord? o Along the same lines, it would illuminate the issue, if the nations receive “נַחֲלָה” elsewhere, since God makes “them” (nations or God’s people) return to their “נַחֲלָה”.

-> Further terms (e.g., "יְרֻשָּׁה") are included subsequently to include other instances, which might reflect the nations receiving an inheritance/possesion (which consequently might be restored to them).
