Description

This query looks for potential "true discontinuous phrases" according to the definition of true discontinuous phrases that typically occur in non-configurational languages, i.e., a noun followed by a discontinuous modifier (adjective, demonstrative, or numeral). Note that Exo 26:26 and 2 Chr 7:10 should not be among the results of this query. The analysis of these verses has recently been corrected, but this is not yet visible in the 2017-version of the database used to publish queries.

Query instruction (mql)

select all objects where

[phrase focus
   [word sp=subs]
   !
   [word sp<>subs]*
   [gap
      NOTEXIST[clause rela IN (Attr, RgRc)]
   ]
   [word sp=art]*
   [
   [word sp=adjv AND NOT (st=c AND ls=ordn) AND NOT (ls=gntl AND pdp=subs)]
   OR
   [word sp=prde]
   ]
]
OR
[phrase focus
   [word sp=subs]
   !
   [word sp<>subs]*
   [gap
      NOTEXIST[clause rela IN (Attr, RgRc)]
   ]
   [phrase_atom
      [word FIRST sp=subs AND ls=card]
      NOTEXIST[word pdp=subs AND ls<>card]
   ]
]
