Description

Deze query zoekt naar alle wayyiqtol vormen in de Psalmen. Voor het de presentatie heb ik de focus ook op de voorafgaande waw gezet, zodat het geheel gearceerd is. Het totaal van 666 is daarom een dubbele telling. In totaal komen er 333 wayyiqtols in de Psalmen voor.
