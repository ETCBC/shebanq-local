Description

In the query I want to have all the cases where God is the subject and PQD is the predicate. The research question behind this query is: Does the statement that God visitits people occure more often in the Bible book Genesis. That is why I search for all the cases where this verb is used with the subject God or JHWH. In Genesis 21 the qal vorm is used. That is why I only want the predicate in the qal form.

Query instruction (mql)

SELECT ALL OBJECTS WHERE

[book book = Genesis                    //The verses need to be in the book Genesis
[clause 
[unorderedgroup                         //The order of the different words does not matter
 [phrase Focus function = Pred]         //The predicate needs to be the verb PQD in the qal
      [word focus lex = "PQD["  AND vs = qal]
   [phrase FOCUS function = Subj]        //The subject needs to be JHWH or Elohim
    [word focus lex = "JHWH/" OR lex = ">LHJM/"]
]
]
