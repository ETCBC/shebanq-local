Description

In Lev 18:25. 28a and 20:22 we find the figurative expression of the land ‘vomiting/‌spitting out’ its inhabitants. Do we find this metaphor elsewhere? \[Possibly in Jer 25:27 (see note), where ‘all the kingdoms’ are instructed to “drink, get drunk and vomit” (cf. the instances of the noun QJ>/ קיא/ ‘vomit’ in Isa 19:14 and Jer 48:26); other instances of this verb have animate subjects.\] And how should we interpret the qal form in v.28b? \[As “de-causative”(?) ‘to be vomited out’ (hif ‘to cause so. to be vomited out’).\]

Note: In Jer 25:27 the same root is written without an א and it is tagged as a separate lexeme QJH[, though its semantics are the same (HALOT, for instance, treats them in one lemma; see also Bauer-Leander, Historische Grammatik, §59p, archive.org; Bergsträsser, II: §28t, archive.org).
