Description

This query intends to find all occasions where God/YHWH/Adonai is the object of the verb 'to bless' (Hebrew: ברך) rather than the subject.

This query searches for any clause that contains the verb 'to bless' and an object phrase containing 'God/YHWH/Adonai'. Since the order of the object and the predicate within the clause is irrelevant, I used [UnorderedGroup.

The "NOTEXIST" condition filters is optional. Without it, the query gives two unwanted results: 2 Samuel 21:3 ("the inheritance of YHWH") and Psalm 115:13 ("the 'fearers' of YHWH"). In both cases, YHWH is not the object, but the nomen rectum part of the genitive construction that constitutes the object.

Run with the "NOTEXIST" condition, the query leaves out those two problematic verses, by filtering out those clauses that contain object phrases with God/YHWH/Adonai as the nomen rectum part of a genitive construction. Unfortunately, this leaves out Genesis 24:48 as well, because its object phrase (YHWH, the God of my Lord Abraham) contains both YHWH as the object of the verb 'to bless' and, in an apposition, God and Adonai as the nomen rectum (which the NONEXIST condition filters out).

To see how English Bible Translations have dealt with those cases where God is the object of blessing, I have compared the NIV (New International Version) and the KJV (King James Version). Interestingly, the KJV translates ‘bless the LORD’, while the NIV translates ‘praise the LORD’. In some cases, for example, in Job, both the KJV and the NIV translate ‘curse God’.

Query instruction (mql)

select all objects where
   [clause
      [UnorderedGroup
         [word FOCUS sp = verb AND lex = "BRK["
         ]
         [phrase function = Objc
//The NOTEXIST condition is optional (see note)
            NOTEXIST [subphrase rela = rec
            ]
            [word FOCUS lex = ">LHJM/" OR lex = "JHWH/" OR lex = ">DNJ/"
            ]
         ]
      ]
   ]
