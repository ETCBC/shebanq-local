Description

Waar komt in de Pentateuch voor dat de engel van de Heer een vrouw vindt en/of tegen haar spreekt?

Om dit te onderzoeken moeten zowel de engel van de Heer als de vrouw waar hij iets aan doet of tegen zegt in een enkelvoudige zin te vinden zijn in een narratieve zin. Omdat ik breed wil zoeken wat de engel doet of zegt, laat ik vrij wat het werkwoord is.

Engel van de Heer is: ML>K/ JHWH/ of ML>K/ >LHJM/. Vrouw moet voorkomen in de suffix van het werkwoord of in het complement van de zin.

Query instruction (mql)

select all objects where

[verse book IN (Genesis, Exodus, Leviticus, Numeri, Deuteronomium)
    [clause domain = 'N'
        [word FOCUS prs_gn = f]
        [word lex = 'ML>K/']
        ..
        [word lex = 'JHWH/' OR lex = '>LHJM/']
    ]
OR
    [clause domain = 'N'
        [word lex = 'ML>K/']
        ..
        [word lex = '>LHJM/' OR lex = 'JHWH/']
        [phrase function = Cmpl
            [word FOCUS gn = f]
        ]
    ]
