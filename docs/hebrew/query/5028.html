Description

Search all cases in which any name of God (אֵל, אֱלֹהִים, יְהוָה) and Jeremiah appear in one clause. Make sure that you find both orders: Jeremiah=>God-name, God-name=>Jeremiah. Also make sure that you capture both orthographic variations of the name Jeremiah (יִרְמְיָה, יִרְמְיָהוּ). How often do God and Jeremiah appear in one clause?
