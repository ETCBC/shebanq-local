Description

In 1 Sam 16:14 we read about “an evil spirit from God”. Is this reference unique, or are there other references of an evil spirit of God/the Lord”?

Note 1: The references of an evil spirit of God/the Lord is only unique if we look to the word order.

Note 2: If you search not search on clause but on phrase level the case of Judges 9:23 will be missed. In that particular case the evil spirit is not directly linked to God but with the help of a verb (שלח).

Note 3: To use the OR operator for the different designations of God, an additional clause-function has been added within the UnorderedGroup.

Note 4: I have searched for three designations of God but you can add more designations of God.
