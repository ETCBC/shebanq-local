Description

The Old Testament contains the Tetragrammaton יהוה written with different vowels, for example with the vowels of of אֲדֹנַי (Adonai, ETCBC transliteration: >:ADON@J).

Query

The regexp in the query is to select for the fowels for adOnAi, transliterated as O and @, which should be around the Wav. The regexp includes '.*' to allow for in-between cantillation marks (which are also included in feature g_word as digits).

Notes

See also similar query in following Text-Fabric NoteBook with overview of the various vocalisations of the Tetragrammaton.

Query instruction (mql)

select all objects where
[clause
   [word FOCUS g_cons='JHWH' AND g_word~'O.*W.*@']
// strictly speaking the regexp should also include the trailing 'A'. However, this does not provide any results.
]
