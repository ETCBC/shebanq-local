Description

In summary, this query is designed to find all verses in the Hebrew Bible where a single clause contains the verbal stem in the Nifal or Pual for "give birth" (JLD[), followed at some point later in the clause by the preposition "L" (L).

Query instruction (mql)

select all objects where
[verse
  [clause
   [word focus lex = 'JLD[' AND vs = pual] 
  OR
   [word FOCUS lex = 'JLD[' AND vs = nif] 
  ..
  [word focus lex = "L"]
  ]
]
