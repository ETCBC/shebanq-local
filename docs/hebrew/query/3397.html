Description

Met behulp van Shebanq wordt er met een Query zoekt naar alle gevallen waarin God afwezig is in het Bijbelboek 1&2 Samuel. Dit naar aanleiding van de afwezigheid van de Godsnaam in het Bijbelboek. Door middel van deze Query kan er onderzocht worden of het vaak voorkomt dat in het Bijbelboek Samuel in sommige hoofdstukken niet voor komt.

Query instruction (mql)

Select all objects where
[chapter FOCUS book = Samuel_I OR book = Samuel_II
   NOTEXIST [word lex = "JHWH/"]
   NOTEXIST [word lex = ">LWH/"]
]
