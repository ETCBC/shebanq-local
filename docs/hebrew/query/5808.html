Description

This query search for any construct chains that are governed by a personal pronoun. The expectation is that this does not exist, as Biblical Hebrew tends to mark pronominal genitive relations with a suffix (מַלְכִּי) or occasionally לְ (as מֶ֫לֶךְ לִי, which is actually a non-verbal sentence).

It turns out that there is only one instance of a personal pronoun governing a construct chain (Nah 2:9).
