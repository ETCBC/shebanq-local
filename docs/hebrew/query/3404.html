Description

In I Samuel 28:21 zegt de vrouw uit Endor dat ze 'haar ziel in haar handpalm heeft geplaatst'. Wat bedoelde ze hier mee? M.b.v. deze query kunnen we kijken naar de nuances van iets in iemands handpalm (KP/) plaatsten (FJM[). Daarna kunnen we beredeneren wat het zou betekenen als een ziel (NPC/) in iemands handpalm wordt geplaatst.
