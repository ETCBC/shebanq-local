Description

This query looks in Jeremiah for all locational phrases consisting of:
• a subphrase containing one of the nouns מקום 'place', ארץ 'land',
  עיר 'city' and בית 'house', directly followed by
• a subphrase with a demonstrative relation to its mother.
(It is possible to explicitly specify that the first subphrase should be the mother of the second, but this does not change the results.)

Query instruction (mql)

Select all objects where

[book book IN (Jeremia)
   [phrase
      [subphrase FOCUS
         [word lex='MQWM/' OR lex='>RY/' 
          OR lex='<JR/' OR lex='BJT/']
      ]
      [subphrase FOCUS rela=dem]
   ]
