Description

Selects all instances of the hif'il of נצב, and their objects if present, by looking for:

- a participle or predicate (complement) (with object/subject suffix) phrase with the hif'il of נצב and an object phrase within the same clause (both phrases in FOCUS) (19 cases);
- a participle or predicate (complement) (with object/subject suffix) phrase with the hif'il of נצב without an object phrase (2 cases);

NB: if the phrase with נצב is a participle with object suffix phrase, any additional object phrase is most likely not the object of the participle. (However, there are 0 of these cases).

Query instruction (mql)

Select all objects where

[clause
   [phrase FOCUS function IN (PrcS,Pred,PreC,PreO,PreS,PtcO)
      [word lex="NYB[" AND vs=hif]
   ]
   ..
   [phrase FOCUS function=Objc]
   OR
   [phrase FOCUS function=Objc]
   ..
   [phrase FOCUS function IN (PrcS,Pred,PreC,PreO,PreS,PtcO)
      [word lex="NYB[" AND vs=hif]
   ]
]
OR
[clause
   [phrase FOCUS function IN (PrcS,Pred,PreC,PreO,PreS,PtcO)
      [word lex="NYB[" AND vs=hif]
   ]
   NOTEXIST [phrase function=Objc]
]
