Description

shibboleth versus sibboleth

This query looks for the word shibboleth in combination with the non word sibboleth. The word shibboleth was being used as a password to separate friend from foe. The foes were the ones pronouncing sibboleth, the friends said shibboleth. The story mentions that 42,000 Ephraimites were killed this way.
