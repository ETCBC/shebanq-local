Description

This query lists all the times there is mention of 'approaching the tabernacle' or 'approaching the tent of meeting' or 'approaching the tent of testimony'. First, we searched for the times when there is mention of the noun 'draw near' in relation to the tent/tabernacle. Then a search was made on the times when there is mention of the verb 'approach' in relation to the tent/tabernacle. This query aims to investigate whether one can also approach the tent without fear of dying. The query returns 18 results in seven verses.

Query instruction (mql)

select all objects where
//First search on the nouns approach and tabernacle/tent of meeting/tent of testimony.
[Clause
  [phrase function = PreC
    [word FOCUS lex = 'QRB=/']
  ]
  ..
  [phrase function = Cmpl
    // Tabernacle
    [word FOCUS lex = 'MCKN/']
    or
    // Tent of Meeting
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = 'MW<D/']
    or
    // Tent of Testimony
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = '<DWT/']
  ]
]
//Then search on the verb approach in combination with tabernacle/tent of meeting/tent of testimony.
or
[clause
  [phrase
    [word FOCUS pdp = verb
            and lex = 'QRB['
    ]
  ]
  ..
  [phrase function = Cmpl
    // Tabernacle
    [word FOCUS lex = 'MCKN/']
    or
    // Tent of Meeting
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = 'MW<D/']
    or
    // Tent of Testimony
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = '<DWT/']
  ]
