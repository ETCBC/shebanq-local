Description

Usually when speaking about blessing (Hebrew: ברך) people think of an act by God towards people, but can God also be the object of this verb?

Note 1: Because SHEBANQ is searching in the 'chronological order' of a query you can use the OR operator and manually add the different word orders. I used 'UnorderedGroup' so that the order of the words in the clause does not matter.

Note 2: I have searched for three designations of God but you can add more designations of God.

Note 3: In the case of Genesis 24.48 the Christian Standard Bible translates ברך with blessing and the International Standard Version is using the word praising as translation. In, for instance, Judges 5.9 both translations are using the word blessing.
