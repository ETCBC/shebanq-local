Description

In 1 Sam 16:14 we read about “an evil spirit from God”. Is this reference unique, or are there other references of an evil spirit of God/the Lord”?

ANSWER: An other way to search for words in a different order is by the use of UNORDEREDGROUP (instead of the use of OR and the different variations of order). Because UNORDEREDGROUP does not work with OR in it, it is needed to search for the Names of God in a 'new' phrase. Because this clause is 'within' the first written phrase, the boundaries result to be the same. When the phrases are changed into clauses, also Judges 9:23 will be found. Only in this case it is not clear if the evil spirit is from God, the evil spirit in Judges 9:23 is only sent by God.
