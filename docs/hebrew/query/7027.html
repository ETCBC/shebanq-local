Description

Deze query zoekt naar een woord in de status constructus met een 'univalent final' (d.w.z., een he, waw, of yod aan het einde van het woord) gevolgd door een woord in de status constructus. Middels deze query kan men onderzoeken hoe een status constructus onderbroken kan worden, en wat de mogelijke oorzaak van deze onderbreking is (bijv. oude naamvalsresten).
