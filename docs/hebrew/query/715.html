Description

Finding all locations in the tenach where נא is preceded by an imperative and followed up by a prolonged imperative in the next clause.

De zegening zoals deze in Genesis 27 wordt gedaan is een bijzonder ritueel waar nog niet alles over bekend is. Uit het taalgebruik kan informatie over de mate van gezag worden afgeleid. Bij de aanbieding van de maaltijd door Jakob worden specifieke vormen gebruikt, Jakob gebruikt een imperatief gevolgd door een 'smeekpartikel' met daarop een verlengde imperatief. Izaak gebruikt precies dezelfde vormen in vers 27, het past dus bij dit ritueel. Met deze query wordt er naar precies deze combinatie van vormen gezocht door heel het Oude Testament. Wat deze combinatie aan gezag vertoont op andere plaatsen in het O.T. kan ook van toepassing zijn op deze rituele zegening.
