Description

This query looks for instances where an evil spirit is sent from the LORD. The preposition "from" is absent from the query to make sure all verses are found.

Other than the I Sam 16-19 narrative with the evil spirit coming to Saul, there is only one other instande of an evil spirit coming from the LORD (Judges 9).
