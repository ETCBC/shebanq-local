Description

To which divine name is the property ascribed of ‘having brought the people out of Egypt’ in the Hebrew Bible? To answer this question, I made a query that sought for all such occurrences in the Hebrew Bible. The query had the following criteria: (1) it should search for several divine names; (2) it should search for a verb that indicates the activity of bringing / leading the people out of Egypt; (3) it should search for the word Egypt. Scope of the query: The query searches for three divine names and three verbs for ‘leading’. The three ‘objects’ of the query may appear in any random order. Limit of the query: Other divine names and verbs are excluded. And the three ‘objects’ of the query need occur in one clause.
