Description

Het woord (leger)kamp komt meerdere keren voor in het vertaalde hoofdstuk (1 Sam 17). We zien door middel van de query dat dit woord niet heel vaak voorkomt in het boek, maar wel dat dit altijd voorkomt in combinatie met de strijd met de Filistijnen. Eigenlijk is dit wel vanzelfsprekend, aangezien je in oorlog altijd meerdere kampen hebt, anders is er geen oorlog. Achteraf is dit motiefwoord mijns inziens dus niet heel erg veelzeggend.
