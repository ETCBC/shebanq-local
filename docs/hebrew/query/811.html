Description

A more advanced version of 'Comparatives': looks for comparative constructions (both attributive and predicative) by collecting phrase atoms, the first lexeme of which has pdp 'adjv', and which is specified by a phrase that starts with the lexeme MN.

Note that this query needs refining: On the one hand, its results also include non-comparative constructions (e.g. 1Kgs 11:26), while on the other hand it misses cases which are comparative, but which have been parsed differently.

Query instruction (mql)

select all objects where
[phrase FOCUS
   [phrase_atom as S
      [word first pdp = adjv]
   ]
   ..
   [phrase_atom mother = S.self
      [word first lex = "MN"]
   ]
]
