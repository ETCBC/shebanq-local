Description

This query is to demonstrate that the majority of occurrences of the construction DBR + <L are with a pronominal suffix. Compare this with occurrences of DBR + <L

Dr. Glanz query -

select all objects where [clause [UnorderedGroup [phrase function IN (Pred, PreO, PreS) [word FOCUS lex='DBR['] ] [phrase function IN (Cmpl) [word FOCUS lex='<L' AND prs_ps IN (p1, p2, p3)] ] ] ]
