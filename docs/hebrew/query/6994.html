Description

This query lists all the times there is mention of 'approaching/come to/go to the tabernacle' or 'come to/go to/approaching the tent of meeting' or 'come to/go to/approaching the tent of testimony'. In the vers 28 of Numbers 17, there is used the adjective of the word approaching. For the verbs 'come' and 'go' I couldn't this form in de Hebrew Bible. First, we searched for the times when there is mention of the noun 'draw near' in relation to the tent/tabernacle. Then a search was made on the times when there is mention of the verb 'approach' in relation to the tent/tabernacle. This query aims to investigate whether one can also approach the tent without fear of dying. The query returns 82 results in 29 verses.

Query instruction (mql)

select all objects where
//First search on the nouns approach and tabernacle/tent of meeting/tent of testimony.
[Clause
  [phrase function = PreC
    //Approaching
    [word FOCUS lex = 'QRB=/']
  ]
  ..
  [phrase function = Cmpl
    // Tabernacle
    [word FOCUS lex = 'MCKN/']
    or
    // Tent of Meeting
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = 'MW<D/']
    or
    // Tent of Testimony
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = '<DWT/']
  ]
]
//Then search on the verb approach in combination with tabernacle/tent of meeting/tent of testimony.
or
[clause
  [phrase
    [word FOCUS pdp = verb
           and (lex = 'QRB=/' //approaching
             or lex = 'BW>['  //come
             or lex = 'HLK[' // go
               )
    ]
  ]
  ..
  [phrase function = Cmpl
    // Tabernacle
    [word FOCUS lex = 'MCKN/']
    or
    // Tent of Meeting
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = 'MW<D/']
    or
    // Tent of Testimony
    [word FOCUS lex = '>HL/']
    [word FOCUS lex = '<DWT/']
  ]
]
